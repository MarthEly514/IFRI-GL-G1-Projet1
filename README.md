# IFRI-GL-G1-Projet1 -- CampusDocs
**CampusDocs** est une application informatique faite en **Java** (SpringBoot et JavaFx) permettant de gérer efficacement les actes administratifs au sein des universités notamment IFRI, en respectant les principes du génie logiciel et en suivant la méthode agile Scrum.


## Prerequis:
JDK 21+
JavaFX (client)
SpringBoot (serveur)
